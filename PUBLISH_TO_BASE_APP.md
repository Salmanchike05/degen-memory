# Как отправить Degen Memory в Base App

Приложение уже в Farcaster и открывается в превью. Чтобы оно появилось в **Base App**, сделайте следующее.

---

## 1. Опубликовать через пост в Base App

По документации Base, приложение попадает в каталог, когда вы **публикуете пост с URL приложения** в Base App.

1. Откройте **Base App** (в браузере или в приложении).
2. Создайте **новый пост** (cast).
3. Вставьте в пост ссылку на приложение:
   ```
   https://degen-memory.vercel.app/
   ```
4. Опубликуйте пост.

После публикации ссылка превратится в embed с кнопкой «Play Degen Memory», и приложение станет доступно из Base App.

---

## 2. (Опционально) Проверить в Base Build

Если хотите привязать приложение к аккаунту разработчика и видеть его в панели:

1. Откройте: **https://build.base.org/** (или **https://www.base.dev/preview**).
2. В разделе **Account association** вставьте URL: `https://degen-memory.vercel.app/`.
3. Нажмите **Verify** и при необходимости обновите `accountAssociation` в `src/minikit.config.ts` (у вас он уже заполнен).

---

## 3. Превью перед постом

Перед тем как постить, можно ещё раз проверить:

- **https://base.dev/preview** — вставьте URL приложения и убедитесь, что превью и кнопка запуска работают.

---

## Итог

- **Минимум для появления в Base App:** пост в Base App с URL `https://degen-memory.vercel.app/`.
- **Уже есть:** деплой на Vercel, manifest, account association, работающее превью.
- После поста приложение будет открываться из Base App по этой ссылке.
